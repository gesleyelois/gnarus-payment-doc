# gnarus-payment-doc

Wiki de modelagem incremental para um sistema de pagamento.

## Estrutura

- `schema.sql`: schema base da versao 1
- `docs/index.html`: wiki visual do dominio
- `docs/assets/app.js`: renderizacao das secoes e dos diagramas
- `docs/assets/styles.css`: estilos da wiki
- `AGENTS.md`: regras permanentes para agentes

## Regra do projeto

Comecar simples e evoluir por etapas. Toda mudanca de regra deve atualizar a wiki e o schema juntos.

Abra `docs/index.html` em um navegador para visualizar a wiki.
